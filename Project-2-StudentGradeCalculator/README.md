# DecodeLabs Internship Projects

# Project 2 — Student Grade Calculator

A console-based Java application that calculates a student's total marks, average percentage, and grade.
- Takes marks (out of 100) for any number of subjects (1-20)
- Calculates total marks and average percentage
- Assigns grades (A+ to F) based on percentage
- Per-subject and overall Pass/Fail indicator
- Input validation for numbers and ranges



\*\*Tech:\*\* Java (Core), Scanner I/O, Loops, Conditionals

# Grading Scheme

| Average % | Grade |

|-----------|-------|

| >= 90     | A+    |

| >= 80     | A     |

| >= 70     | B     |

| >= 60     | C     |

| >= 50     | D     |

| >= 40     | E     |

| < 40      | F     |
