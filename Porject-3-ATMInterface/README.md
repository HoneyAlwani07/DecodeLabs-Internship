# Project 3 — ATM Interface
A console-based Java application that simulates an ATM system using Object-Oriented Programming (OOP) principles, ensuring secure financial transactions and data encapsulation.

- PIN-based authentication with 3-attempt lockout mechanism
- Deposit and withdraw funds with balance and limit validations
- Check balance and view full transaction history with timestamps
- Input validation for negative amounts, insufficient funds, and ATM note multiples (₹100)
- Data encapsulation (private balance and PIN, accessed via public methods)
- Tech: Java (Core), OOP (Classes, Encapsulation), Scanner I/O, Loops, Conditionals
